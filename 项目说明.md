# Felis Code — 公式ポートフォリオサイト

## 概要

神戸拠点の個人フルスタック開発スタジオ「Felis Code」の公式ポートフォリオ兼コンタクトサイト。
静的サイト（HTML / CSS / Vanilla JS）で構築されており、GitHub Pages にてホスティング。
外部フレームワーク・ビルドツールを一切使わず、`tailwindcss` CDN のみを利用したシンプルな構成。

- **URL**: <https://felis-code.github.io>
- **言語**: 日本語（一部英語）
- **ホスティング**: GitHub Pages（`main` ブランチ直接デプロイ）

## 主な実装・仕様

### テーマ

- ライトモード / ダークモード を CSS クラスで切り替え
- CSS カスタムプロパティ（`--brand-green: #45b059` など）でブランドカラーを管理

### Projects カルーセル

- **方式**: クローンスライドによる無限ループ（clone-last + 実3枚 + clone-first = 計5枚）
- **自動再生**: 10秒間隔、マウスオーバーで一時停止
- **操作**: 左右矢印ボタン / ドットインジケーター / タッチスワイプ / キーボード ←→
- **UI**: 上部プログレスバー + `1 / 3` カウンター
- **連打対策**: `isAnimating` フラグでアニメーション中の操作をブロック

### コードウィンドウアニメーション

- `felis_web_stack.vue` / `scheduler_core.py` の両カードにコードスクロール演出
- CSS `@keyframes codeScroll`（40s）+ JS による内容複製でシームレスループ
- タイトルバーへの溢れ防止のため、スクロール領域を `overflow-hidden` 内部ラッパーで分離

### スクロールアニメーション

- `IntersectionObserver` で各要素が視野に入ったタイミングで `fadeInUp` アニメーション発火
- `data-animate` / `data-delay` 属性でスタガー制御

### Services → Projects リンク

- Services カードの「詳しく見る →」クリック時、対応するカルーセルスライドへ自動移動

## 開発時の注意

- **ローカルプレビュー**: `file://` で直接開くと anchor リンクにセキュリティ警告が出る場合あり。VS Code の Live Server 拡張（`http://localhost:5500`）推奨。
- **OGP画像**: `ogp.png`（1200×630px 推奨）をルートに配置すると SNS シェア時にサムネイルが表示される。
- **メールアドレス**: Contact セクションの `mailto:` は `luping263@gmail.com` に設定済み。
- **価格**: 「相談・見積もりは無料」として掲載。固定料金は設定していない。
